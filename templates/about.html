About

covidcases.com is a website for finding information about local instances of the 2019-nCoV disease. It pulls from the JHU CSSE COVID data, available here.


This website is for informational purposes only. It depends on external data sources and does not represent medical advice. Currently, only state-level resolution is available for the US.


All code is available from GitHub. Star this project if you find it to be helpful! If you have any questions or concerns, contact me.
